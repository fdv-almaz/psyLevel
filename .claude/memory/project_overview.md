**Project:** psyLevel — новое многоплатформенное мобильное приложение

**Tech Stack:** Flutter (Dart) для iOS и Android

**Core Functionality:**
- Пользователь вводит пульс/ЧСС (BPM) и диастолическое артериальное давление/ДАД (mmHg)
- Приложение рассчитывает уровень стресса по формуле: **Stress = (1 - (ДАД / ЧСС)) × 100**
- Результат выражается как процент (0-100 шкала)
- Простое, интуитивное приложение для быстрого ввода данных и просмотра результатов

**Target Platforms:** iOS + Android

**Current Status:** Проект в начальной стадии, каталог структурирован, но код еще не написан
